PROBLEM STATEMENT: <br>
---
Build a Countdown timer using Python and graphic user interface that helps users to set a duration of time to complete their tasks which helps to manage time with productivity.<br>

SCOPE OF THE PROJECT:<br>
---
1.Provides a Graphic user interface(GUI) for simple interaction.

2.Validates task and time input.

3.Execute a countdown timer using background threading.

4.Notifies the user when the timer completes.

TARGET USERS:
---
1.Students managing study sessions or assignments.

2.Professionals timing tasks during work.

3.Anyone needing a simple,distraction free countdown timer.

HIGH LEVEL FEATURES:<br>
---
1.Task name and time duration input.

2.Error handling and validation for incorrect inputs.

3.Smooth countdown timer with GUI updates.

4.Completition alert using popup notifications
