# VITYARTHI-PROJECT_COUNTDOWN-TIMER
OVERVIEW :
---
The project is a simple yet functional Countdown Timer which is built using python.It allows users to input their task and the duration needed for it and then tracks the remaining time with a clear and simple user friendly display <br>

FEATURES :
---
1.User Management Module<br>
2.Task Input and Processing<br>
3.Countdown Timer Module<br>
4.Graphical User Interface(GUI)<br>
           
TECHNOLOGIES/TOOLS USED:<br>
---
1.Python - Programming Language<br>
2.Tkinter- GUI<br>
3.Threading module(python)- Thread support<br>
4.Tkinter messagebox - Time handling<br>

STEPS TO INSTALL AND RUN THE PROJECT:<br>
---
1.Install Python<br>
2.Download and clone the Project<br>
3.Navigate to the Project Folder<br>
4.Run the application<br>

INSTRUCTIONS FOR TESTING:<br>
---

2.Test Time Input Validation<br>

4.Test Task Display<br>
